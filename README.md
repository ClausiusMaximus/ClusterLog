# ClusterLog

Eine installierbare Progressive Web App (PWA) zum Erfassen und Auswerten von Clusterkopfschmerz-Attacken.

## Ziele

- Offline nutzbar
- Schnell bedienbar
- Lokale Datenspeicherung
- Statistik
- CSV- und JSON-Export
