# ClusterLog

ClusterLog ist eine Progressive Web App (PWA) zum schnellen Erfassen von Clusterkopfschmerz-Attacken.

## Funktionen

- 📅 Datum & Uhrzeit automatisch
- ⏱ Dauer im Format MM:SS bzw. HH:MM:SS
- 🔥 KIP-Skala 0–10
- 💾 Speicherung in IndexedDB
- 📱 Installierbare PWA
- 🌙 Dark Mode
