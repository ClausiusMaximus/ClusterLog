# ADR-001

## Titel

Feature-basierte Architektur

## Status

Akzeptiert

## Entscheidung

ClusterLog wird feature-basiert aufgebaut.

## Begründung

Alle Dateien einer Funktion liegen zusammen.

Dadurch bleibt das Projekt auch bei vielen Features übersichtlich.

## Konsequenzen

Positiv

- bessere Wartbarkeit
- klare Struktur
- gute Skalierbarkeit

Negativ

- etwas mehr Ordner
