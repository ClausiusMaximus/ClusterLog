# UI Guidelines

## Zielgruppe

Menschen mit Clusterkopfschmerzen.

Während einer Attacke muss die Bedienung möglichst einfach sein.

---

## Grundregeln

- große Buttons
- hoher Kontrast
- wenig Text
- große Schrift
- maximal drei Schritte bis zur Eingabe

---

## Navigation

Bottom Navigation.

---

## Farben

Primärfarbe

Blau

Sekundärfarbe

Türkis

---

## Design

Material Design 3

---

## Bedienung

Alle wichtigen Funktionen sollen mit einer Hand erreichbar sein.
